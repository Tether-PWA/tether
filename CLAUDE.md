# Tether

PWA accountability partner app.
Single file: index.html
Font: Space Grotesk (Google Fonts)
Primary color: #1B2E4B (navy)
Background: #F7F8FA (off-white)
Max width: 480px centered
No dark mode.
No frameworks. Vanilla JS only.
All data stored in localStorage.
Voice notes use MediaRecorder API.
Browser Notification API for reminders.

## Deploy
Run: ./deploy-te.sh
Or type: deploy-te in terminal after alias is set up.

## Backup
Auto-saves 10 latest versions to /backups/ folder.
